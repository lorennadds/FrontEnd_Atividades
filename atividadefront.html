Material de Estudo Digital - Vetores em Linguagem C

O que são Vetores?

Vetores (arrays) são estruturas de dados usadas para armazenar vários valores do mesmo tipo em uma única variável.

Cada posição do vetor possui um índice que começa em 0. Isso significa que o primeiro elemento sempre estará na posição 0.

Características dos Vetores

  • Armazenam vários valores do mesmo tipo
  • São acessados através de índices
  • O índice começa em 0
  • São muito usados com estruturas de repetição

Declaração de um vetor

int v[5];

Nesse exemplo criamos um vetor chamado v com 5 posições, capaz de armazenar números inteiros.

Exemplo em Linguagem C

#include <stdio.h>

int main(){

int v[5];

v[0] = 10;
v[1] = 20;

printf("%d", v[0]);

return 0;
}

Usando Vetores com FOR

int i;
int v[5];

for(i = 0; i < 5; i++){
    v[i] = i;
}

O laço for é frequentemente utilizado para percorrer ou preencher vetores.
